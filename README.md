# 👕 تطبيق متجر الملابس (Clothing API)

مرحباً بك في مشروع متجر الملابس الشامل! هذا المشروع يوفر حلاً متكاملاً لإدارة متجر ملابس أونلاين مع واجهة أمامية جميلة وخادم خلفي قوي.

## 📋 نظرة عامة على المشروع

هذا المشروع يتكون من ثلاثة أجزاء رئيسية:
- **الخادم الخلفي (Backend)**: مكتوب بـ Python يدير البيانات والعمليات
- **قاعدة البيانات (Database)**: تخزين معلومات المنتجات والطلبات والعملاء
- **الواجهة الأمامية (Frontend)**: تطبيق ويب تفاعلي لعرض المنتجات والتسوق

---

## ✨ المميزات الرئيسية

### 1. **إدارة المنتجات** 🛍️
- عرض جميع المنتجات المتاحة
- تصنيف المنتجات (قمصان، بناطيل، فساتين، إلخ)
- البحث والتصفية حسب الفئة والسعر
- عرض تفاصيل كل منتج

### 2. **سلة التسوق** 🛒
- إضافة المنتجات إلى سلة التسوق
- تعديل الكمية
- حذف المنتجات
- حفظ السلة تلقائياً (LocalStorage)
- حساب الإجمالي تلقائياً

### 3. **إدارة الطلبات** 📦
- تتبع الطلبات
- عرض سجل الطلبات
- حالة الطلب (قيد المعالجة، شُحن، مكتمل)

### 4. **إدارة العملاء** 👥
- تسجيل عملاء جدد
- تسجيل الدخول والخروج
- حفظ معلومات العميل
- سجل الطلبات الشخصي

### 5. **الدفع والفاتورة** 💳
- معالجة الدفع
- إنشاء فواتير تلقائية
- تحميل الفواتير

## 🚀 البدء السريع

### المتطلبات
- Python 3.8 أو أحدث
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- SQL Database (SQLite, MySQL أو PostgreSQL)

### التثبيت

#### 1. استنساخ المشروع
```bash
git clone https://github.com/BahaaMo245205/clothing_API.git
cd clothing_API
```

#### 2. إنشاء بيئة افتراضية
```bash
python -m venv venv
# على Windows
venv\Scripts\activate
# على Mac/Linux
source venv/bin/activate
```

#### 3. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

#### 4. تشغيل الخادم الخلفي
```bash
python backend/app.py
# أو حسب اسم ملف التطبيق الخاص بك
```

#### 5. فتح الواجهة الأمامية
```bash
# افتح المتصفح وانتقل إلى:
http://localhost:5000
# أو ملف index.html مباشرة
```

## 📁 شرح مفصل لمجلدات المشروع

### 📂 **frontend_new/** (الواجهة الأمامية الجديدة)
الواجهة الأمامية الحديثة المكتوبة بـ HTML و CSS و JavaScript:
- **index.html**: الصفحة الرئيسية مع جميع الأقسام
- **styles.css**: تصميم استجابي جميل ومتوافق مع الهواتف
- **script.js**: وظائف التفاعل والتسوق

### 📂 **backend/** (الخادم الخلفي)
يحتوي على:
- API endpoints لجميع العمليات
- معالجة قاعدة البيانات
- المصادقة والأمان
- معالجة الطلبات والدفع

### 📂 **database/** (قاعدة البيانات)
- ملفات تعريف قاعدة البيانات
- SQL queries المهمة
- سكريبتات التهيئة

### 📂 **ConnactSQL/** (الاتصال بقاعدة البيانات)
- أدوات الاتصال بـ SQL
- كلاسات للتعامل مع البيانات

### 📂 **Clothings_Photo/** (صور المنتجات)
- صور جميع المنتجات
- صور عالية الجودة

## 💻 وظائف API الرئيسية

### الحصول على المنتجات
```http
GET /api/products
```
استجابة:
```json
{
  "products": [
    {
      "id": 1,
      "name": "قميص كلاسيكي",
      "price": 29.99,
      "category": "shirts"
    }
  ]
}
```

### إضافة طلب
```http
POST /api/orders
Content-Type: application/json

{
  "customer_id": 1,
  "items": [{"product_id": 1, "quantity": 2}],
  "total": 59.98
}
```

### الحصول على الطلبات
```http
GET /api/orders/{customer_id}
```
